# TYQbag Skills

Personal agent skills published for installation with the Skills CLI.

## Available Skills

### `test-strategy`

Use this skill to decide what testing evidence is needed before, during, and after a code change. It covers risk-based test decisions, test layer selection, regression planning, and completion verification with real evidence.

Install from this repository:

```sh
npx skills add TYQbag/TYQbag-skills --skill test-strategy
```

Preview available skills:

```sh
npx skills add TYQbag/TYQbag-skills --list
```
